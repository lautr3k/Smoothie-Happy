    public command(settings: Object): Promise source 

    import command from 'smoothie-happy/src/command.js'

    Sends an arbitrary command to the board over HTTP protocol.

    • This function is based on the axios library, please read the axios documentation if you wish to go further.
    • The [ method, url, data, timeout ] params can not be overwritten in axios config param.

    Params:

    NameTypeAttributeDescription
    settings Object

    command settings
    settings.address string

    ip or hostname (without protocle http://)
    settings.command string

    raw command (without trailing LF char)
    settings.timeout number
    • optional
    • default: 0

    connexion timeout
    settings.axiosConfig Object
    • optional
    • default: {}

    axios config (documentation )

    Return:

    Promise

    return a Promise with the axios response

    Example:

    command({
  address: '192.168.1.121',
  command: 'version'
})
  .then(response => {
    console.log(response)
  })
  .catch(error => {
    console.log(error)
  })

    See:

    • https://github.com/axios/axios#axiosconfig

    public ls(settings: Object): Promise source 

    import ls from 'smoothie-happy/src/commands/ls.js'

    Send ls command.

    Params:

    NameTypeAttributeDescription
    settings Object

    command settings
    settings.address string

    ip or hostname (without protocle http://)
    settings.path string
    • optional
    • default: '/sd'

    path to list
    settings.getSize boolean
    • optional
    • default: false

    fetch files size (slower)
    settings.recursive boolean
    • optional
    • default: false

    fetch inner folders (slower)
    settings.onFile function
    • optional
    • default: null

    called when a file is found
    settings.timeout number
    • optional
    • default: 0

    connexion timeout
    settings.axiosConfig Object
    • optional
    • default: {}

    axios config (documentation )

    Return:

    Promise

    Example:

    ls({
  address: '192.168.1.121',
  path: '/sd',
  getSize: true,
  recursive: true,
  onFile (file) {
    console.log('file:', file)
  }
})
  .then(response => {
    console.log(response)
  })
  .catch(error => {
    console.log(error)
  })
