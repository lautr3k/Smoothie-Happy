    NameTypeAttributeDescription

    data Mixed
    • optional
    • default: null

    Data to send with the request.
    headers Object
    • optional
    • default: {}

    Headers to send with the request.
    timeout Integer
    • optional
    • default: 5000

    Timeout for this request in milliseconds.
    maxAttempts Integer
    • optional
    • default: 5

    Maximum number of attemps before rejection.
    attemptDelay Integer
    • optional
    • default: 5000

    Delay in milliseconds between two attemps.
    mimeType String
    • optional
    • default: 'text/plain'

    Expected response mime type.
    on Object
    • optional
    • default: {}

    Collection of function that takes as single parameter an RequestEvent instance. Ex.: { 'download.load': event => { }, 'upload.progress': event => { } } See src/request/event/types.js for possible values.
    filters Object
    • optional
    • default: {}

    Collection of filter functions.
    filters.status Function
    • optional

    A function that takes as single parameter the request status code and return an boolean to resolve/reject the request.
    filters.response Function
    • optional

    A function that takes as single parameter the request response and return a new version or throw an Error to reject the Promise.
